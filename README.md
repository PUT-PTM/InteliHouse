# InteliHouse
## Overview
Intelihouse project using epaper display and communicating over WiFi 
## Tools
STM32F407G-DISC1 
Waveshare 1.54 inch E-ink Display Module with SPI Interface
Waveshare 8 Push Buttons
## How to run
BUSY -> PE11
RST  -> PE9
DC   -> PE8
CS   -> PE10
CLK  -> PB13
DIN  -> PB15
K0 -> PB4
K1 -> PB5
K2 -> PB7
K3 -> PB8
